# TECH STACK + FOLDER STRUCTURE

---

## Stack

| Layer | Technology | Why |
|---|---|---|
| Framework | React 18 + TypeScript | Component model matches screen-based prototype |
| Build | Vite 5 | Fast, PWA plugin available |
| Styling | Tailwind CSS + CSS Variables | Utility-first + custom garage theme vars |
| State | Zustand | Lightweight, no boilerplate |
| Database | Supabase (PostgreSQL) | Free tier, real-time, Auth included |
| Auth | Supabase Auth (email/password) | Single mechanic login |
| i18n | i18next + react-i18next | 3 languages, 120 keys |
| PWA | vite-plugin-pwa | Service worker, manifest, installable |
| Routing | React Router v6 | Screen-to-screen navigation |
| Icons | lucide-react | Matches prototype SVG icons |
| Date | date-fns | Relative time ("3 din pehle"), IST formatting |

## Install Commands

```bash
npm create vite@latest snbc-app -- --template react-ts
cd snbc-app
npm install

# Core deps
npm install @supabase/supabase-js zustand react-router-dom
npm install i18next react-i18next i18next-browser-languagedetector
npm install date-fns lucide-react

# Dev deps
npm install -D tailwindcss postcss autoprefixer
npm install -D vite-plugin-pwa

# Init Tailwind
npx tailwindcss init -p
```

## TypeScript Types (`src/types/index.ts`)

```typescript
export type VehicleStatus = 'in_repair' | 'done' | 'paid';
export type PaymentType = 'full' | 'partial';
export type PaymentMethod = 'upi' | 'cash' | 'bank';
export type VehicleType = 'Scooter' | 'Bike';
export type Language = 'en' | 'hi' | 'gu';

export interface Vehicle {
  id: string;                       // UUID from Supabase
  customer_name: string;
  customer_whatsapp: string;        // 10 digits, no +91
  vehicle_type: VehicleType;
  number_plate: string;             // always UPPERCASE
  owner_name: string | null;
  owner_whatsapp: string | null;    // 10 digits, no +91
  problem: string;
  estimate: number;                 // integer rupees
  delivery_by: string | null;       // free text
  status: VehicleStatus;
  created_at: string;               // ISO timestamp
  updated_at: string;
}

export interface Payment {
  id: string;
  vehicle_id: string;
  amount_paid: number;              // integer rupees
  total_amount: number;             // copy of estimate at time of payment
  payment_type: PaymentType;
  payment_method: PaymentMethod;
  note: string | null;
  paid_at: string;                  // ISO timestamp
}

export interface GarageSettings {
  garage_name: string;
  mechanic_name: string;
  mechanic_whatsapp: string;        // 10 digits вЂ” receives self-copy WA messages
  upi_id: string | null;            // shown in WA payment messages
}
```

## i18next Setup (`src/i18n/index.ts`)

```typescript
import i18n from 'i18next'
import { initReactI18next } from 'react-i18next'
import en from './en.json'
import hi from './hi.json'
import gu from './gu.json'

const savedLang = localStorage.getItem('snbc_lang') || 'en'

i18n
  .use(initReactI18next)
  .init({
    resources: { en: { translation: en }, hi: { translation: hi }, gu: { translation: gu } },
    lng: savedLang,
    fallbackLng: 'en',
    interpolation: { escapeValue: false }
  })

export default i18n
// To change language: i18n.changeLanguage('hi'); localStorage.setItem('snbc_lang', 'hi')
```
